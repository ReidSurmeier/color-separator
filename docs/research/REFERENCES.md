# Reference Sites

## riso.lfesp.me (PRIMARY UI reference)
- SvelteKit app, Tailwind CSS
- Upload image → halftone dot separation into riso ink colors
- Right sidebar: color swatches that slide out on hover, showing color name
- Main area: canvas preview with paper texture overlay (mix-blend-screen)
- 4 sliders: dot size, offset strength, strength, texture
- Color swatches shown as large clickable blocks below sliders
- Clean, minimal, white background, lowercase aesthetic
- "Choose File" button at bottom
- Output: halftone canvas rendering

## dithering.ksawerykomputery.pl (INTERACTION reference)
- p5.js canvas app
- Floating nav panel (top-left, white bg) with controls
- Parameters: size (W×H), source (camera/video/screen/image), color palette, FPS, scale, blur, hue, saturation, brightness, contrast
- Color swatches as small 20x20 color inputs with remove button
- Sliders for all parameters
- Preview toggles: fullscreen, fill, hue cycle, randomize size
- Render: video/frame export
- Dark background, pixelated canvas rendering

## teenage.engineering OP-1 module graphics
- SVG line illustrations: intricate circuit-board style paths
- Two-tone: dark blue (#272F66) and bright blue (#698EFF)
- Animated "waveform" style — city skylines made of signal paths
- Use as animated header/logo decoration for the woodblock tool
